## 1. Create folder and virtual environment 
```
python -m venv venv
```
___
## 2. Activate virtual environment
```
venv/Scripts/Activate
```
___
## 3. Install aiogram and openai
```
pip install aiogram
pip install openai
pip install googletrans==3.1.0.a0 Pillow 
```
